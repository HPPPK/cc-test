referenceId: workflow:memory-update
description: Record final workflow handoff and reusable project context without changing production files.
user-invocable: false
---

Use this skill when a workflow finish phase needs durable project notes and handoff artifacts.

- Summarize completed scope, validation evidence, preview status, known risks, and next workflow options.
- Update only workflow-owned handoff or project-context artifacts allowed by the phase.
- Do not start follow-up workflows or edit production source during memory update.
