Use this skill when a workflow phase needs Codex to convert the approved plan into a small, ordered Todo list.

- Read the current workflow artifacts and user constraints before adding or changing todos.
- Keep todos scoped to the confirmed batch and mark only evidence-backed progress complete.
- Do not edit production files from a planning-only phase.
