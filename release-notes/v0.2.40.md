# Claude Code Jiangxia v0.2.40

## Default workflow packs

- Windows and macOS desktop installers again include the three current default workflow ZIP packs.
- On first launch, the app copies the default packs into the local workflow-pack directory automatically.
- When a later app version contains a newer default pack with the same internal workflow ID, the local old pack is replaced even when its ZIP filename differs.
- User-created workflow ZIPs with different workflow IDs remain untouched.

## macOS packaging

- The macOS desktop sidecar now receives the packaged workflow-pack resource directory, so the same first-run and ID-based update behavior works from the DMG installation.
