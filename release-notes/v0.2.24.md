# cc-test v0.2.24

## Sidebar project-menu layering

This release fixes the project context menu in the desktop sidebar being obscured by other UI layers.

### Highlights

- The sidebar project context menu is rendered through `createPortal(..., document.body)` so it is placed in the global UI layer.
- The project menu stacking level is raised from `z-50` to `z-[9999]`.
- A focused Sidebar test assertion verifies the menu uses the intended high stacking class.

## Upgrade notes

- No workflow ZIP/template, runtime tool, permission, or workflow-routing behavior changed in this release.
- Windows installers and the GitHub Release are built remotely by GitHub Actions from tag `v0.2.24`.
