# v0.2.38

## macOS automatic updates
- 正式发布工作流新增 Apple Silicon macOS 构建，与 Windows 使用同一套 Tauri updater 签名密钥。
- Release 同时上传 macOS DMG、updater 压缩包和签名，并把 macOS 平台记录合并进 `latest.json`。
- Windows 与 macOS 发布任务串行更新清单，避免并发覆盖导致任一平台的自动更新条目丢失。

## Release reliability
- 发布前显式校验 updater 私钥，并固定上传 updater 清单与签名产物。
- 修正 Release 资源命名配置，使 GitHub Action 使用当前支持的参数。
- 补充跨平台发布结构回归测试，确保 Windows 与 Apple Silicon macOS 条目持续存在。

## macOS path compatibility
- 专家材料读取使用真实路径进行边界校验，兼容 macOS `/var` 与 `/private/var` 的系统路径别名，同时继续阻止目录逃逸。
- 放宽流式 SDK 集成测试的慢速环境截止时间，减少覆盖率运行中的误报。
